Preliminary / Dry Tests
Colour of Salt (Observation)
Blue → Cu²⁺ (CuSO₄·5H₂O)
Green → Fe²⁺ (pale green) or Cr³⁺ or Ni²⁺
Yellow/orange → Fe³⁺ or Cr⁶⁺ (CrO₄²⁻)
Violet/dark purple → KMnO₄ or I₂
Pink → Co²⁺ or Mn²⁺
Black → CuS, FeS, Ag₂S, MnO₂, CuO, PbS, CoS, NiS
White → most common (Na⁺, Ca²⁺, Ba²⁺, Al³⁺, Zn²⁺, Pb²⁺)
Red → Pb₃O₄ (minium), Hg₂I₂, Cu₂O, HgI₂ (above 127°C orange)
Yellow → As₂S₃, CdS, Na₂CrO₄, PbCrO₄, SnS₂, AgI, PbI₂
Orange → Sb₂S₃, Cr₂O₇²⁻
Brick-red → Cu₂O, Fe₂O₃
Heating / Action of Heat
CaCO3 → CaO + CO2↑ (CO₂ turns lime water milky)
Na2CO3 → does not decompose easily on heating
MgCO3 → MgO + CO2↑
2KNO3 → 2KNO2 + O2↑ (glowing splint reignites)
4KNO3(strong heat) → 2K2O + 4NO2 + O2
2Pb(NO3)2 → 2PbO + 4NO2↑(brown) + O2↑
2AgNO3 → 2Ag + 2NO2↑ + O2↑ (leaves grey Ag)
2Hg(NO3)2 → 2HgO + 4NO2↑ + O2↑
2HgO → 2Hg(vapour) + O2↑ (black → silver drops on tube)
4Fe(NO3)3 → 2Fe2O3 + 12NO2↑ + 3O2↑ (brown gas)
NH4Cl → NH3↑ + HCl↑ (sublimes, white ring on cool part)
NH4NO3 → N2O + 2H2O (laughing gas)
NH4NO2 → N2 + 2H2O (on gentle heating)
FeS + O2(heat) → FeO + SO2↑ (sulphur smell)
CuSO4·5H2O → CuSO4(white) + 5H2O (blue → white, reversible)
FeSO4·7H2O → FeSO4(white) then → Fe2O3 + SO3 (smells acid)
Flame Test
Li → crimson red
Na → golden yellow (persistent)
K → lilac / pale violet (seen through blue glass)
Rb → red-violet
Ca → brick red
Sr → scarlet/crimson red
Ba → apple green / yellowish green
Cu → blue-green (without HCl) / emerald green (with HCl)
Pb → pale blue / whitish
As, Sb → pale blue
Borax Bead Test
Na2B4O7·10H2O → Na2B4O7 → 2NaBO2 + B2O3 (molten bead)
CoO + B2O3 → Co(BO2)2 (blue, both flames)
CuO + B2O3 → Cu(BO2)2 (green ox / red reducing)
Fe2O3 + B2O3 → Fe(BO2)3 (yellow-brown ox / green red)
MnO + B2O3 → Mn(BO2)2 (amethyst/violet ox)
Cr2O3 + B2O3 → 2Cr(BO2)3 (green both flames)
NiO + B2O3 → Ni(BO2)2 (brown ox)
Anion Analysis — Group I (Dil. H₂SO₄)
Carbonate (CO₃²⁻) / Bicarbonate (HCO₃⁻)
Na2CO3 + H2SO4(dil) → Na2SO4 + H2O + CO2↑ (brisk effervescence)
CO2 + Ca(OH)2 → CaCO3↓(white, milky) + H2O (lime water test)
CaCO3 + CO2 + H2O → Ca(HCO3)2 (soluble) (excess CO₂ — milkiness disappears)
2NaHCO3 + H2SO4(dil) → Na2SO4 + 2H2O + 2CO2↑
Sulphite (SO₃²⁻)
Na2SO3 + H2SO4(dil) → Na2SO4 + H2O + SO2↑ (pungent, suffocating smell)
SO2 + K2Cr2O7(acidified) → Cr2(SO4)3 (orange → green, confirmatory)
SO2 + 2H2S → 3S + 2H2O (bleaches)
BaCl2 + Na2SO3 → BaSO3↓(white, soluble in HCl) (distinguishes SO₃²⁻ from SO₄²⁻)
Sulphide (S²⁻)
Na2S + H2SO4(dil) → Na2SO4 + H2S↑ (rotten egg smell)
H2S + Pb(CH3COO)2(paper) → PbS↓(black) (confirmatory)
H2S + 2AgNO3 → Ag2S↓(black) + 2HNO3
H2S + CdSO4 → CdS↓(yellow) + H2SO4
Na2S + Na2[Fe(CN)5NO] → Na4[Fe(CN)5NOS] (purple/violet — sodium nitroprusside test)
Nitrite (NO₂⁻)
NaNO2 + H2SO4(dil) → NaHSO4 + HNO2
3HNO2 → HNO3 + 2NO↑ + H2O (colorless gas → brown in air)
FeSO4 + HNO2 + H2SO4 → [Fe(NO)]SO4(brown ring)
KMnO4(acidified) + 5KNO2 + 3H2SO4 → 2MnSO4 + 5KNO3 + K2SO4 + 3H2O (decolorises KMnO₄)
Anion Analysis — Group II (Conc. H₂SO₄)
Chloride (Cl⁻)
NaCl + H2SO4(conc) → NaHSO4 + HCl↑ (colourless pungent fumes)
HCl + NH3(gas) → NH4Cl↑(white fumes) (hold NH₃ rod — white smoke)
Cl− + AgNO3 → AgCl↓(white, curdy) + NO3−
AgCl + 2NH3 → [Ag(NH3)2]Cl(soluble) (soluble in dil NH₃ — distinguishes from AgBr/AgI)
MnO2 + 4HCl → MnCl2 + Cl2↑ + 2H2O (green-yellow Cl₂, bleaches)
Bromide (Br⁻)
NaBr + H2SO4(conc) → NaHSO4 + HBr↑; HBr + H2SO4 → Br2↑(brown fumes) + SO2
Br− + AgNO3 → AgBr↓(pale yellow) (partially soluble in conc NH₃, insoluble in dil NH₃)
Br− + Cl2(water) → Br2(orange layer in CCl4/CHCl3)
Iodide (I⁻)
NaI + H2SO4(conc) → NaHSO4 + HI↑; 2HI + H2SO4 → I2(violet fumes) + SO2 + 2H2O
I− + AgNO3 → AgI↓(yellow) (insoluble in both dil and conc NH₃)
I2 + starch → blue-black complex (most sensitive I₂ test)
I− + Cl2(water) → I2(violet layer in CCl4)
2KI + Pb(NO3)2 → PbI2↓(golden yellow) + 2KNO3
Nitrate (NO₃⁻)
NaNO3 + H2SO4(conc) → NaHSO4 + HNO3↑ (vapours turn Cu foil green)
Brown Ring Test: FeSO4 + NO → [Fe(NO)]SO4 (brown ring) in presence of conc H2SO4
NO3− + Fe²⁺ + 4H+ → Fe³⁺ + NO↑ + 2H2O; NO + FeSO4 → [Fe(NO)]SO4 (brown ring at interface)
Diphenylamine test: NO3− + diphenylamine + H2SO4 → deep blue colour
Acetate (CH₃COO⁻)
CH3COONa + H2SO4(conc) → CH3COOH↑ (vinegar smell)
CH3COONa + C2H5OH + H2SO4 → CH3COOC2H5 + H2O (fruity smell — esterification)
3CH3COO− + FeCl3 → [Fe(CH3COO)3](blood red, hydrolyses on boiling → brown ppt)
Anion Analysis — Group III (Special Reagents)
Sulphate (SO₄²⁻)
Ba²⁺ + SO42− → BaSO4↓(white) (insoluble in dil HCl and HNO₃ — confirms SO₄²⁻)
BaCl2 + Na2SO4 → BaSO4↓ + 2NaCl
BaSO3 + 2HCl → BaCl2 + SO2↑ + H2O (BaSO₃ dissolves in HCl, BaSO₄ does not)
Phosphate (PO₄³⁻)
Na3PO4 + 3AgNO3 → Ag3PO4↓(yellow) + 3NaNO3
PO43− + 12MoO42− + 24H+ + 3NH4+ → (NH4)3[P(Mo3O10)4]↓(canary yellow) (ammonium molybdate test)
PO43− + Mg²⁺ + NH4+ + NH3 → MgNH4PO4↓(white, crystalline)
Oxalate (C₂O₄²⁻)
Na2C2O4 + H2SO4(dil) → Na2SO4 + H2C2O4 → 2CO2↑ + H2O (on warming)
C2O42− + Ca²⁺ → CaC2O4↓(white) (insoluble in acetic acid, soluble in HCl)
5C2O42− + 2MnO4− + 16H+ → 10CO2 + 2Mn²⁺ + 8H2O (decolorises KMnO₄)
Borate (BO₃³⁻) / Borax
Na2B4O7 + H2SO4(dil) → Na2SO4 + 4H3BO3
H3BO3 + CH3OH + H2SO4 → B(OCH3)3 (burns with green flame — borate test)
Chromate/Dichromate (CrO₄²⁻ / Cr₂O₇²⁻)
CrO42− + Ba²⁺ → BaCrO4↓(yellow, soluble in HCl not in CH3COOH)
Cr2O72− + H2O2 + H+ → CrO5(blue, in ether) (perchromic acid test)
2CrO42− + 2H+ → Cr2O72− + H2O (yellow → orange)
Cation Analysis — Group 0 · NH₄⁺
NH4+ + NaOH(hot) → NH3↑ + Na+ + H2O (turns moist red litmus blue)
NH3 + HCl → NH4Cl (white fumes — confirm NH₄⁺)
NH4+ + Nessler's reagent → Hg2INH2↓(brown ppt)
Cation Analysis — Group I (dil. HCl) · Pb²⁺, Ag⁺, Hg₂²⁺
Precipitating Reagent: dil HCl
Ag+ + Cl− → AgCl↓(white, curdy)
AgCl + 2NH3 → [Ag(NH3)2]Cl (dissolves in NH₃) → acidify → AgCl reppt
Hg22+ + 2Cl− → Hg2Cl2↓(white)
Hg2Cl2 + 2NH3 → HgNH2Cl(white) + Hg↓(grey-black) (turns grey-black)
Pb2+ + 2Cl− → PbCl2↓(white)
PbCl2(soluble in hot water) → Pb²⁺ + 2Cl⁻; + 2I⁻ → PbI2↓(golden yellow, on cooling)
Pb²⁺ + CrO42− → PbCrO4↓(yellow) (confirmatory for Pb²⁺)
Pb²⁺ + H2SO4(dil) → PbSO4↓(white) + 2H+
Cation Analysis — Group II (H₂S in Acid) · Hg²⁺, Pb²⁺, Bi³⁺, Cu²⁺, Cd²⁺, As³⁺, Sb³⁺, Sn²⁺
Group IIA (insoluble in yellow ammonium sulphide) — Hg²⁺, Pb²⁺, Bi³⁺, Cu²⁺, Cd²⁺
Hg²⁺ + H2S → HgS↓(black)
Pb²⁺ + H2S → PbS↓(black)
Bi³⁺ + H2S → Bi2S3↓(black)
Bi³⁺ + Na[Sn(OH)3] → Bi↓(black) (Bi confirmatory — stannite reduction)
Cu²⁺ + H2S → CuS↓(black)
Cd²⁺ + H2S → CdS↓(yellow)
Group IIB (soluble in yellow ammonium sulphide) — As³⁺, Sb³⁺, Sn²⁺
As³⁺ + H2S → As2S3↓(yellow)
As2S3 + (NH4)2S2 → (NH4)3AsS4 (soluble thioarsenate)
Sb³⁺ + H2S → Sb2S3↓(orange)
Sn²⁺ + H2S → SnS↓(brown/dark brown)
Sn⁴⁺ + H2S → SnS2↓(yellow)
SnCl2 + 2HgCl2 → Hg2Cl2↓(white) + SnCl4 (Sn²⁺ confirmatory)
SnCl2(excess) + Hg2Cl2 → 2Hg↓(grey-black) + SnCl4
Cation Analysis — Group III (NH₃ / NH₄Cl) · Fe³⁺, Al³⁺, Cr³⁺
Fe³⁺ + 3OH− → Fe(OH)3↓(reddish-brown)
Al³⁺ + 3OH− → Al(OH)3↓(white, gelatinous)
Al(OH)3 + excess NaOH → [Al(OH)4]−(soluble) (Al³⁺ amphoteric)
Al³⁺ + aluminon(aurintricarboxylic acid) → red lake (confirmatory)
Cr³⁺ + 3OH− → Cr(OH)3↓(green)
Cr(OH)3 + excess NaOH → [Cr(OH)4]−(soluble, green)
Cr(OH)3 + H2O2 + 4NaOH → 2Na2CrO4(yellow) + H2O (Cr³⁺ confirmatory)
Na2CrO4 + BaCl2 → BaCrO4↓(yellow, soluble in HCl)
Fe³⁺ + SCN− → [Fe(SCN)]²⁺ (blood red — most imp. Fe³⁺ test)
Fe³⁺ + K4[Fe(CN)6] → Fe4[Fe(CN)6]3↓(Prussian blue)
Cation Analysis — Group IV (H₂S in Alkaline) · Zn²⁺, Mn²⁺, Co²⁺, Ni²⁺
Zn²⁺ + H2S(NH3 medium) → ZnS↓(white)
ZnS dissolves in dil HCl (distinguishes from CdS)
Zn²⁺ + K4[Fe(CN)6] → K2Zn3[Fe(CN)6]2↓(white) (confirmatory)
Zn²⁺ + dithizone → pink-red complex (extra confirmation)
Mn²⁺ + H2S → MnS↓(cream/buff coloured)
Mn²⁺ + NaBiO3 + HNO3 → MnO4−(purple/violet) (Mn confirmatory)
Mn²⁺ + 2NH3 + 2H2O → Mn(OH)2↓(white) → MnO2(brown, on standing in air)
Co²⁺ + H2S → CoS↓(black)
Co²⁺ + 4SCN− → [Co(SCN)4]²⁻(blue in amyl alcohol) (confirmatory Co)
Co²⁺ + 2NH4SCN + excess → blue complex in organic layer
Ni²⁺ + H2S → NiS↓(black)
Ni²⁺ + 2DMG(dimethylglyoxime) + 2NH3 → [Ni(DMG)2]↓(rose red) (most imp. Ni confirmatory)
Borax bead Ni → brown (ox flame)
Cation Analysis — Group V (NH₄)₂CO₃ · Ca²⁺, Sr²⁺, Ba²⁺
Ca²⁺ + CO32− → CaCO3↓(white)
Ca²⁺ + (NH4)2C2O4 → CaC2O4↓(white, insoluble in CH3COOH) (confirmatory Ca)
Sr²⁺ + CO32− → SrCO3↓(white)
Sr²⁺ + (NH4)2SO4 → SrSO4↓(sparingly soluble, distinguishes from Ba)
Ba²⁺ + CO32− → BaCO3↓(white)
Ba²⁺ + CrO42− → BaCrO4↓(yellow, soluble in HCl but not in CH3COOH) (Ba confirmatory)
Ba²⁺ + SO42− → BaSO4↓(white, insoluble in HCl) (confirms both Ba²⁺ and SO₄²⁻)
Flame: Ca = brick red | Sr = scarlet | Ba = apple green
Cation Analysis — Group VI · Mg²⁺, Na⁺, K⁺
Mg²⁺ + 2NH3 + HPO42− → MgNH4PO4↓(white, crystalline) (confirmatory Mg)
Mg²⁺ + magneson reagent (NaOH) → blue lake (confirmatory)
Mg²⁺ + 2NaOH → Mg(OH)2↓(white, gelatinous)
Na⁺ → golden yellow flame test
Na⁺ + zinc uranyl acetate → NaZn(UO2)3(CH3COO)9↓(yellow crystalline) (Na confirmatory)
K⁺ → lilac/violet flame (seen through blue cobalt glass)
K⁺ + Na3[Co(NO2)6] → K2Na[Co(NO2)6]↓(yellow) (sodium cobaltinitrite — K confirmatory)
K⁺ + ClO4− → KClO4↓(white) (perchlorate test)

Solubility Rules (JEE Key Points)
All nitrates are soluble
All acetates are soluble (except basic acetates)
All chlorides soluble EXCEPT AgCl, PbCl₂, Hg₂Cl₂
All sulphates soluble EXCEPT BaSO₄, SrSO₄, PbSO₄ (insoluble); CaSO₄ (sparingly soluble)
All carbonates insoluble EXCEPT Na₂CO₃, K₂CO₃, (NH₄)₂CO₃
All sulphides insoluble EXCEPT Na₂S, K₂S, (NH₄)₂S, BaS, CaS, MgS
All hydroxides insoluble EXCEPT NaOH, KOH, Ba(OH)₂ (soluble); Ca(OH)₂ (sparingly soluble)
