GROUP 13 — BORON & ALUMINIUM
B · Al · Ga · In · Tl
R01–R22
▸ BORON (B) — Borax · Boric Acid · Diborane · BX₃
R01
Borax → Boric Acid (acidification)
Na2B4O7 + H2SO4 + 5H2O → Na2SO4 + 4H3BO3
Borax = Na₂B₄O₇·10H₂O; boric acid = weak monobasic acid; Lewis acid (accepts OH⁻)
R02
Borax Bead Test — Coloured Metaborates
Na2B4O7 Δ → 2NaBO2 + B2O3 (glassy bead) + MO → M(BO2)2 (coloured)
Cu → blue/green; Co → deep blue; Ni → grey; Fe → yellow/brown; Mn → violet
R03
Boric Acid — Lewis Acid Action with Water
H3BO3 + H2O ⇌ [B(OH)4]– + H+
Does NOT donate H⁺; it accepts OH⁻ from water — acts as Lewis acid, not Brønsted acid
R04
Boric Acid → Metaboric Acid → Boron Trioxide
H3BO3 373K → HBO2 + H2O ;   HBO2 red hot → B2O3 + H2O
R05
Diborane Preparation (Lab)
3LiAlH4 + 4BF3 ether → 2B2H6 + 3LiF + 3AlF3
Also: 2NaBH₄ + I₂ → B₂H₆ + 2NaI + H₂
R06
Diborane Hydrolysis
B2H6 + 6H2O → 2H3BO3 + 6H2
R07
Diborane + NH₃ → Borazine (Inorganic Benzene)
B2H6 + 2NH3 Δ → B3N3H6 + 3H2
Borazine (borazole) — isoelectronic with benzene; structure alternates B–N bonds
R08
Diborane + O₂ — Combustion
B2H6 + 3O2 → B2O3 + 3H2O     (ΔH = −2165 kJ mol⁻¹)
R09
BCl₃ Hydrolysis (vs AlCl₃)
BCl3 + 3H2O → H3BO3 + 3HCl
BCl₃ hydrolyses completely (no d-orbitals to back-bond); AlCl₃ gives Al(OH)₃ or Al₂O₃
R10
BF₃ + F⁻ → Tetrafluoroborate (Lewis Acid-Base)
BF3 + F– → [BF4]–   (sp³, tetrahedral)
▸ ALUMINIUM (Al) — Extraction · Amphoteric · Alum · AlCl₃
R11
Al + Dil H₂SO₄
2Al + 3H2SO4(dil) → Al2(SO4)3 + 3H2↑
R12
Al — Passivation with Conc. HNO₃
Al + Conc. HNO3 → No reaction (passivation; oxide layer)
R13
Al Amphoteric — Reaction with NaOH
2Al + 2NaOH + 2H2O → 2NaAlO2 + 3H2↑
Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O (oxide also amphoteric)
R14
Alum — Preparation & Formula
Al2(SO4)3 + K2SO4 + 24H2O → K2SO4·Al2(SO4)3·24H2O
Potash alum = KAl(SO₄)₂·12H₂O; used as mordant; Al³⁺ in [Al(H₂O)₆]³⁺
R15
Thermite Reaction
Fe2O3 + 2Al Δ → Al2O3 + 2Fe   (ΔH = −852 kJ)
Aluminothermy — used to weld railway tracks; Al reduces metal oxide
R16
AlCl₃ Hydrolysis
AlCl3 + 3H2O → Al(OH)3 + 3HCl   (fumes in moist air)
R17
Al(OH)₃ — Amphoteric
Al(OH)3 + 3HCl → AlCl3 + 3H2O   ;   Al(OH)3 + NaOH → Na[Al(OH)4]
R18
Cryolite Formation (Hall-Héroult)
AlF3 + 3NaF → Na3AlF6 (cryolite)   — solvent for Al₂O₃ in electrolysis
R19
Hall-Héroult — Cathodic & Anodic
Cathode: Al3+ + 3e– → Al   ;   Anode: 2O2– – 4e– → O2   (C anode burns)
R20
Baeyer's Process — Bauxite Purification
Al2O3 + 2NaOH → 2NaAlO2 + H2O → dilute & CO2 → Al(OH)3↓ → calcine → Al2O3
R21
Al + Cl₂
2Al + 3Cl2 → 2AlCl3   (exists as Al₂Cl₆ dimer in solid/vapour)
R22
Al + Fe₃O₄ (Thermite variant)
8Al + 3Fe3O4 Δ → 4Al2O3 + 9Fe

GROUP 14 — CARBON, SILICON, TIN & LEAD
C · Si · Ge · Sn · Pb
R23–R42
▸ CARBON — CO · CO₂ · Carbonate · Diamond · Fullerene
R23
CO — Reducing Agent (Fe₂O₃)
Fe2O3 + 3CO Δ → 2Fe + 3CO2
Blast furnace reaction; CO is principal reductant above 983 K
R24
Water Gas Shift Reaction
CO + H2O Δcat ⇌ CO2 + H2
Used in Bosch process; increases H₂ yield from water gas
R25
CO — Toxic Reaction (Haemoglobin)
Hb(O2)4 + 4CO → Hb(CO)4 + 4O2   (CO binds 200× stronger)
R26
CO + Cl₂ — Phosgene
CO + Cl2 hν / charcoal → COCl2   (phosgene, war gas)
R27
CO₂ + H₂O ⇌ H₂CO₃
CO2 + H2O ⇌ H2CO3 ⇌ H+ + HCO3– ⇌ 2H+ + CO32–
R28
CO₂ + NaOH (excess)
CO2 + 2NaOH(excess) → Na2CO3 + H2O   ;   CO₂ (excess) → NaHCO₃
R29
Na₂CO₃ + CO₂ + H₂O
Na2CO3 + CO2 + H2O → 2NaHCO3
Reverse: 2NaHCO₃ →(Δ) Na₂CO₃ + H₂O + CO₂ (baking soda decomposition)
R30
Solvay Process — Key Step
NaCl(sat.) + NH3 + CO2 + H2O → NaHCO3↓ + NH4Cl   (NaHCO₃ calc. → Na₂CO₃)
R31
C + H₂O — Water Gas
C + H2O 1270K → CO + H2 (water gas / syngas)
R32
CS₂ — Carbon Disulfide Preparation
C + 2S 1200K → CS2
▸ SILICON — SiO₂ · Silicates · SiCl₄ · Silicones
R33
SiO₂ + HF (only acid to attack)
SiO2 + 4HF → SiF4↑ + 2H2O   ;   SiF4 + 2HF → H2SiF6 (hexafluorosilicic acid)
Used to etch glass; SiO₂ does NOT react with HCl, H₂SO₄, HNO₃
R34
SiO₂ + NaOH (fused)
SiO2 + 2NaOH fused → Na2SiO3 + H2O   (water glass)
R35
Na₂SiO₃ + CO₂ + H₂O
Na2SiO3 + CO2 + H2O → H2SiO3↓ + Na2CO3
Silicic acid ppt; stronger acid displaces weaker; CO₂ stronger than H₂SiO₃
R36
SiCl₄ Hydrolysis
SiCl4 + 4H2O → Si(OH)4 + 4HCl   (via d-orbital expansion)
CCl₄ does NOT hydrolyse (no d-orbitals in C); SiCl₄ does (3d available)
R37
Silicon — Preparation of Silicones
(CH3)2SiCl2 + H2O → [(CH3)2Si–O]n (silicone polymer) + HCl
Silicones = organosilicon polymers; water repellent, thermally stable
▸ TIN & LEAD — Sn · Pb Reactions
R38
Sn + Conc. HCl
Sn + 2HCl → SnCl2 + H2   ;   SnCl₂ + 2HCl → H₂[SnCl₄]
R39
PbO₂ — Strong Oxidising Agent
PbO2 + 4HCl → PbCl2 + Cl2 + 2H2O
Pb⁴⁺ is oxidising; Pb prefers +2 due to inert pair effect
R40
Pb + Dil H₂SO₄ — No Reaction
Pb + H2SO4(dil) → PbSO4↓ (insoluble protective layer) → No further reaction
R41
Pb + Conc. HNO₃
Pb + 4HNO3(conc.) → Pb(NO3)2 + 2NO2 + 2H2O
R42
SnCl₂ as Reducing Agent
SnCl2 + 2FeCl3 → SnCl4 + 2FeCl2   ;   SnCl₂ + 2HgCl₂(excess) → SnCl₄ + Hg₂Cl₂↓
SnCl₂ excess → Hg₂Cl₂ further → Hg (black) — used to distinguish Hg²⁺

GROUP 15 — NITROGEN & PHOSPHORUS
N · P · As · Sb · Bi
R43–R78
▸ NITROGEN — N₂ · NH₃ · HNO₂ · HNO₃
R43
Haber Process — NH₃ Synthesis
N2 + 3H2 Fe cat / 450°C / 200 atm ⇌ 2NH3   ΔH = −92 kJ
Promoter: K₂O, Al₂O₃; high P, low T (but low T = slow kinetics); compromise at 450°C
R44
NH₃ — Basic Character (Brønsted)
NH3 + H2O ⇌ NH4+ + OH–     Kb = 1.8×10⁻⁵
R45
NH₃ + HCl → NH₄Cl (White Fumes)
NH3(g) + HCl(g) → NH4Cl(s) ↑ (dense white fumes)
R46
NH₃ — Reducing Action (O₂ excess)
4NH3 + 3O2 Δ → 2N2 + 6H2O   (no catalyst)
R47
Ostwald Process — NH₃ Oxidation
4NH3 + 5O2 Pt/Rh / 800°C → 4NO + 6H2O   ;   2NO + O₂ → 2NO₂   ;   3NO₂ + H₂O → 2HNO₃ + NO
R48
HNO₃ (Dil) + Cu → Blue Solution
3Cu + 8HNO3(dil) → 3Cu(NO3)2 + 2NO↑ + 4H2O
R49
HNO₃ (Conc) + Cu
Cu + 4HNO3(conc.) → Cu(NO3)2 + 2NO2↑ + 2H2O
Conc → NO₂ (brown); Dil → NO (colourless); both nitric acid reactions
R50
HNO₃ + S / C / P (Conc, Non-metals)
S + 2HNO3(conc.) Δ → H2SO4 + 2NO   ;   C + 4HNO₃ → CO₂ + 4NO₂ + 2H₂O   ;   P + 5HNO₃ → H₃PO₄ + 5NO₂ + H₂O
R51
HNO₃ — Passivation of Fe, Al, Cr
Fe / Al / Cr + Conc. HNO3 → No reaction (oxide protective film — passivation)
R52
NO₂ + H₂O → HNO₃ + HNO₂
2NO2 + H2O → HNO3 + HNO2
HNO₂ unstable: 3HNO₂ → HNO₃ + 2NO + H₂O (disproportionation)
R53
N₂O₅ — Anhydride of HNO₃
N2O5 + H2O → 2HNO3   ;   2HNO₃ → N₂O₅ + H₂O (drying with P₂O₅)
R54
Brown Ring Test — NO Detection
FeSO4 + NO → [Fe(NO)]SO4 (brown ring, nitrosoferrous)
NO₃⁻ + 3Fe²⁺ + 4H⁺ → NO + 3Fe³⁺ + 2H₂O → brown ring with FeSO₄
R55
Nitrogen Oxides — Summary of Preparations
N₂O: NH₄NO₃ Δ→ N2O+2H₂O  |  NO: 3Cu+8HNO₃(dil)→3Cu(NO₃)₂+2NO+4H₂O  |  NO₂: Cu+4HNO₃(conc)→NO₂  |  N₂O₅: 2HNO₃+P₂O₅→N₂O₅
▸ PHOSPHORUS — P₄ · PH₃ · PCl₃ · PCl₅ · Oxyacids
R56
P₄ Combustion
P4 + 3O2(limited) → P4O6   ;   P₄ + 5O₂(excess) → P4O10
White phosphorus: reactive, glows in dark (phosphorescence), stored in water
R57
P₄O₆ + H₂O → H₃PO₃ (Phosphorous Acid)
P4O6 + 6H2O → 4H3PO3
H₃PO₃ is dibasic (two OH groups, one P-H); good reducing agent
R58
P₄O₁₀ + H₂O → H₃PO₄ (Orthophosphoric)
P4O10 + 6H2O → 4H3PO4   ;   P₄O₁₀ is strongest known desiccant
R59
PCl₃ — Preparation & Hydrolysis
P4 + 6Cl2(limited) → 4PCl3   ;   PCl₃ + 3H₂O → H₃PO₃ + 3HCl
R60
PCl₅ — Preparation & Hydrolysis
PCl3 + Cl2 → PCl5   ;   PCl₅ + H₂O → POCl₃ + 2HCl   ;   PCl₅ + 4H₂O → H₃PO₄ + 5HCl
PCl₅: sp³d, trigonal bipyramidal; dissociates to PCl₃ + Cl₂ on heating
R61
Phosphine PH₃ — Preparation
P4 + 3KOH + 3H2O Δ → PH3↑ + 3KH₂PO₂
PH₃ = phosphine; very poisonous; ignites spontaneously in air (P₂H₄ traces)
R62
H₃PO₂ (Hypophosphorous) — Strong Reducing Agent
H3PO2 + 2AgNO3 + H2O → 2Ag↓ + H₃PO₃ + 2HNO₃
H₃PO₂ monobasic (one OH, two P–H); strongest reducing oxyacid of P
R63
H₃PO₃ — Disproportionation
4H3PO3 Δ → 3H3PO4 + PH3↑
R64
P — Reaction with Conc. HNO₃
P4 + 20HNO3(conc.) + 8H2O Δ → 4H3PO4 + 20NO2
R65
Pyrophosphoric Acid Formation
2H3PO4 Δ → H4P2O7 + H2O   (tetrabasic; further → HPO₃ metaphosphoric)
▸ MORE NITROGEN — Diazotisation · N-compounds
R66
N₂H₄ (Hydrazine) — Reducing Agent
N2H4 + 2H2O2 → N2 + 4H2O   ;   N₂H₄ + Br₂ → N₂ + 4HBr
R67
NH₄NO₂ — Decomposition
NH4NO2 Δ → N2 + 2H2O   (lab preparation of N₂)
R68
NH₄NO₃ — Decomposition
NH4NO3 ~200°C → N2O + 2H2O   (laughing gas)
R69
N₂ + O₂ — Birkeland-Eyde (Arc)
N2 + O2 electric arc / 3000°C ⇌ 2NO   ΔH = +180 kJ (endothermic)
R70
Nitric Oxide — Dimerisation & Oxidation
2NO + O2 → 2NO2   (rapid at RT)   ;   2NO + Cl₂ → 2NOCl (nitrosyl chloride)
R71
Nitrogen Trihalide NF₃ vs NCl₃
NCl3 + 3H2O → NH3 + 3HOCl   (N acts as Lewis base donor)
NF₃ barely hydrolyses (F withdraws e-density from N); NCl₃ hydrolyses easily
R72
NO + KMnO₄ (acidic) — Oxidation of NO
5NO + KMnO4 + 4H2SO4 → 5NO3– + 2Mn2+ + K+ + 4H2O
R73
Aqua Regia — Gold Dissolution
Au + 4HCl + HNO3 → H[AuCl4] + NO + 2H₂O   (3:1 HCl:HNO₃ by volume)
NO₃⁻ oxidises Au to Au³⁺; Cl⁻ complexes as [AuCl₄]⁻; also dissolves Pt
R74
Sodium Nitrite vs Nitrate — Thermal
2NaNO3 Δ → 2NaNO2 + O2   ;   2NaNO₂ →(Δ) Na₂O + 2NO + ½O₂
R75
HNO₃ — Ring Nitration (Conc H₂SO₄)
HNO3 + H2SO4(conc.) → NO2+(nitronium) + HSO4– + H2O   [electrophile]
R76
As₂O₃ — Amphoteric Oxide
As2O3 + 6HCl → 2AsCl3 + 3H2O   ;   As₂O₃ + 6NaOH → 2Na₃AsO₃ + 3H₂O
R77
Marsh Test — As Detection
As2O3 + 6Zn + 6H2SO4 → 2AsH3↑ + 6ZnSO4 + 3H2O   (arsine gas → As mirror)
R78
Bi(OH)₃ — Basic Only (No Acidic Behaviour)
Bi(OH)3 + 3HCl → BiCl3 + 3H2O   (NO reaction with NaOH — basic only)
Metallic character increases down group 15: N<P<As<Sb<Bi

GROUP 16 — OXYGEN & SULPHUR
O · S · Se · Te · Po
R79–R102
▸ OXYGEN — O₃ · H₂O₂ · Oxides
R79
Ozone — Preparation
3O2 silent electric discharge ⇌ 2O3   ΔH = +142 kJ mol⁻¹
R80
Ozone — Iodide Oxidation (Test)
O3 + 2KI + H2O → I2 + 2KOH + O2   (I₂ turns starch blue)
R81
Ozone + Ag₂S (oxidising action)
Ag2S + O3 → Ag2SO4   (converts sulphide → sulphate — not done by O₂)
R82
H₂O₂ — Preparation (Lab)
BaO2 + H2SO4(dil.) → H2O2 + BaSO4↓   ;   Industrial: 2-ethylanthraquinol + O₂
R83
H₂O₂ as Oxidant — KMnO₄ (Acidic)
2KMnO4 + 5H2O2 + 3H2SO4 → 2MnSO4 + K2SO4 + 5O2 + 8H2O
H₂O₂ acts as reductant here (oxidised to O₂); acts as oxidant with PbS→PbSO₄
R84
H₂O₂ as Reductant — AgNO₃
2AgNO3 + H2O2 → 2Ag↓ + 2HNO3 + O2 (Ag⁺ reduced; H₂O₂ oxidised)
R85
H₂O₂ Decomposition (catalysed)
2H2O2 MnO₂ / I⁻ → 2H2O + O2   (disproportionation; cat. raises rate)
▸ SULPHUR — H₂S · SO₂ · SO₃ · H₂SO₄ · Thiosulphate
R86
H₂S — Acidic + Reducing Agent
H2S + 2FeCl3 → 2FeCl2 + S↓ + 2HCl   ;   H₂S + Cl₂ → S↓ + 2HCl
R87
SO₂ — Bleaching (Reducing)
SO2 + H2O → H2SO3   ;   Coloured matter + H₂SO₃ → Colourless (reversible — not permanent bleach)
SO₂ bleaches by reduction; Cl₂ bleaches by oxidation (permanent)
R88
SO₂ — Reducing Agent (KMnO₄ Decolourises)
2KMnO4 + 5SO2 + 2H2O → K2SO4 + 2MnSO4 + 2H2SO4
R89
SO₂ + Cl₂ → Sulphuryl Chloride
SO2 + Cl2 charcoal → SO2Cl2 (sulphuryl chloride; sp³ pyramidal)
R90
Contact Process — SO₃ Manufacturing
2SO2 + O2 V₂O₅ / 450°C / 1–2 atm ⇌ 2SO3   ΔH = −197 kJ
SO₃ absorbed in 98% H₂SO₄ → oleum (H₂S₂O₇); diluted to give H₂SO₄
R91
H₂SO₄ (Conc) + Cu
Cu + 2H2SO4(conc.) Δ → CuSO4 + SO2↑ + 2H2O
R92
H₂SO₄ (Conc) + C / S / P
C + 2H2SO4(conc.) Δ → CO2 + 2SO2 + 2H₂O   ;   S + 2H₂SO₄ → 3SO₂ + 2H₂O
R93
H₂SO₄ + HI/HBr — Oxidising Action
2HI + H2SO4(conc.) → I2 + SO2 + 2H2O   (H₂SO₄ oxidises HI; doesn't oxidise HCl/HF)
R94
Thiosulphate — Na₂S₂O₃ + I₂
2Na2S2O3 + I2 → Na2S4O6 + 2NaI   (sodium tetrathionate; key iodometric titration)
R95
Na₂S₂O₃ + Cl₂ (Excess Halogen)
Na2S2O3 + 4Cl2 + 5H2O → Na2SO4 + H2SO4 + 8HCl   (oxidised to sulphate)
R96
S + Na₂SO₃ → Thiosulphate
Na2SO3 + S boil → Na2S2O3   (preparation of sodium thiosulphate)
R97
S + Conc. H₂SO₄
S + 2H2SO4(conc.) Δ → 3SO2 + 2H2O
R98
H₂SO₄ (Dil) — Salt Displacement
Na2CO3 + H2SO4(dil.) → Na2SO4 + H2O + CO2↑   ;   BaCl₂ + H₂SO₄ → BaSO₄↓(white insol.) + 2HCl
R99
SO₂ + Br₂(aq) — Decolourisation
SO2 + Br2 + 2H2O → H2SO4 + 2HBr   (red-brown Br₂ decolourised)
R100
Oleum (Fuming H₂SO₄) — Dilution
H2S2O7 + H2O → 2H2SO4   ;   SO₃ + H₂SO₄ → H₂S₂O₇
R101
SF₆ — Inertness (Excess F oxidation)
S + 3F2(excess) → SF6   (sp³d², octahedral; kinetically inert to hydrolysis)
R102
H₂S + SO₂ — Wackenroder Reaction
2H2S + SO2 → 3S↓ + 2H2O   (sulphur milk; used to test SO₂)

GROUP 17 — HALOGENS
F · Cl · Br · I · At
R103–R128
▸ FLUORINE (F₂) — Most Electronegatve; Unique Reactions
R103
F₂ + H₂O — Explosive
2F2 + 2H2O → 4HF + O2   (vigorous; F₂ oxidises water)
Only halogen to oxidise water; Cl₂ disproportionates; Br₂/I₂ do not react
R104
F₂ + NaOH
2F2 + 2NaOH → 2NaF + OF2 + H2O   (gives oxygen difluoride, not hypofluorite)
R105
HF — Special Reactions
HF + SiO2 → SiF4↑ + H2O   ;   2HF + SiF₄ → H₂SiF₆ (glass etching)
HF: weakest hydrohalic acid (H-bonded); stored in wax or PTFE bottles
▸ CHLORINE (Cl₂) — Preparation · Bleach · HOCl · Disproportionation
R106
Cl₂ — Lab Preparation
MnO2 + 4HCl(conc.) Δ → Cl2↑ + MnCl2 + 2H2O
R107
Cl₂ + H₂O — Disproportionation
Cl2 + H2O ⇌ HOCl + HCl   (Cl₂ water acidic; HOCl is oxidising bleach)
R108
Cl₂ + Cold Dil. NaOH
Cl2 + 2NaOH(cold dil.) → NaCl + NaOCl + H2O   (sodium hypochlorite)
R109
Cl₂ + Hot Conc. NaOH
3Cl2 + 6NaOH(hot conc.) → 5NaCl + NaClO3 + 3H2O   (sodium chlorate)
R110
Bleaching Powder — Preparation
Ca(OH)2 + Cl2 → Ca(OCl)Cl   (bleaching powder = calcium chlorohypochlorite; ~35% avail Cl₂)
R111
Bleaching Powder + CO₂ / H₂SO₄
Ca(OCl)Cl + CO2 + H2O → CaCO3 + Cl2 + H2O   ;   + dil. H₂SO₄ → CaSO₄ + 2Cl₂ + H₂O
R112
Cl₂ + NH₃ (Excess)
8NH3(excess) + 3Cl2 → 6NH4Cl + N2
Cl₂ excess + NH₃: NCl₃ + 3HCl (trichloroamine); NH₃ excess → N₂
R113
Cl₂ + Na₂S₂O₃ (Reducing Cl₂)
Na2S2O3 + Cl2 + H2O → Na2SO4 + S↓ + 2HCl   (Cl₂ limited)
R114
KClO₃ Thermal Decomposition
2KClO3 MnO₂ / Δ → 2KCl + 3O2↑   (lab O₂); alone → KClO₄ + KCl
▸ HYDROHALIDES — HF · HCl · HBr · HI
R115
HCl — Lab Preparation
NaCl + H2SO4(conc.) Δ → NaHSO4 + HCl↑   ;   NaHSO₄ + NaCl →(Δ,higher) Na₂SO₄ + HCl
R116
HBr — Preparation (from NaBr + H₂SO₄ fails)
PBr3 + 3H2O → 3HBr↑ + H3PO3   (red P + Br₂ → PBr₃; then + H₂O)
H₂SO₄ cannot be used for HBr/HI as it oxidises them; use H₃PO₄ instead
R117
HI — NaI + H₃PO₄
NaI + H3PO4 Δ → HI↑ + NaH₂PO₄   (H₃PO₄ doesn't oxidise HI)
R118
HI — Reducing Strength
2HI + H2SO4(conc.) → I2 + SO2 + 2H2O   (HI > HBr > HCl as reducing agents)
▸ INTERHALOGENS — ClF₃ · BrF₅ · IF₇ · ICl
R119
ICl + KI — Triiodide Complex
ICl + KI → KCl + I2   ;   ICl acts as Cl⁺ donor (electrophilic halogenation)
R120
Cl₂ + F₂ (Excess Cl) → ClF
Cl2 + F2(eql) → 2ClF   ;   Cl₂ + 3F₂ → 2ClF₃ (T-shaped, sp³d)   ;   BrF₅ (square pyramidal)

R121
ClF₃ Hydrolysis
ClF3 + 2H2O → HClO2 + 3HF   (vigorous; fuming)
▸ OXYACIDS OF HALOGENS — HOCl · HClO₂ · HClO₃ · HClO₄
R122
HOCl — Bleaching/Oxidising
HOCl hν → HCl + [O] (nascent oxygen → oxidises dyes)
Acid strength order: HOCl < HClO₂ < HClO₃ < HClO₄ (more O → more electron withdrawal)
R123
HClO₄ — Preparation
KClO4 + H2SO4(conc.) → HClO4 + KHSO4   (perchloric acid — strongest oxyacid)
R124
Iodine — Reaction with Na₂SO₃
I2 + Na2SO3 + H2O → Na2SO4 + 2HI   (I₂ oxidises sulphite)
R125
I₂ + NaOH (cold) — Hypoiodite
I2 + 2NaOH(cold) → NaI + NaOI + H2O   (unstable, decomposes rapidly to NaIO₃)
R126
Pseudohalide — Reaction of CN⁻
2Ag+ + 2CN– → AgCN↓ → [Ag(CN)2]–   ;   2CN⁻ → (oxidise) → (CN)₂ (cyanogen, like Cl₂)
